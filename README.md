Memer
============

This Chrome Extension searches for meme image comments where users don't want to hotlink images and replaces the text with an image.

## Supported Keywords
	* notsureifserious.jpg
	* citationneeded.jpg
	* quizzicaldog.jpg
	* whynotboth.jpg
	* youdontsay.jpg
	* thisiswhywecanthavenicethings.jpg
	* wtfisthisshit.jpg
	* wat.jpg
	* fistofangrygod.jpg
	* icame.jpg
	* thisisdog.jpg
